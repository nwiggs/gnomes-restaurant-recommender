# 🍽️ GNOMES Restaurant Recommender

A web-based restaurant recommendation system designed to help users discover great places to eat in Dallas, Texas.

This project was developed as part of a team collaboration using GitHub.

---

## 📌 Project Overview

The GNOMES Restaurant Recommender is a simple web application that allows users to explore restaurant options based on available data. It uses a dataset of restaurants to generate recommendations and display results through a clean user interface.

---

## 🚀 Features

- 🔍 Browse and explore restaurants
- 📊 Uses a structured dataset (`restaurants.csv`)
- 🌐 Web interface for displaying results
- ⚡ Fast and simple recommendation logic
- 🧩 Built with Python and web templates

---

## 🛠️ Technologies Used

- Python
- Flask (or similar web framework)
- HTML/CSS (Templates)
- CSV dataset for restaurant data
- GitHub for collaboration

## ⚙️ How to Run the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/gnomes-restaurant-recommender.git

2. Navigate into the project folder:

cd gnomes-restaurant-recommender   

3. Install dependencies (if needed):

pip install flask

4. Run the application

python app.py

5. Open your browser and go to:

http://127.0.0.1:5000/
